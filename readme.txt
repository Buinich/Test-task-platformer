Статические данные
Настройки игрока, урон шипов, восстановление аптечек и местонахождения спавнеров хранятся в папке _Project/Resources/StaticData/(PlayerConfig для игрока, Levels для остального) и используются в StaticDataService.

Bootstrap
В проекте используется Zenject. BootstrapInstaller находится в папке _Project/Resources, на ProjectContext.
